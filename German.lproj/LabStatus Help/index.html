LabStatus

LabStatus ist ein kleines Monitoring-Utility um wiederkehrende Kommandos auf eine Menge ähnlich konfigurierter Hosts auzuführen. Sie verwalten beispielsweise ein Labor mit 20 Rechnern. Jeder Rechner kann Windows oder für administrative Aufgaben Linux booten. Wenn Sie nun wissen wollen, welcher Host hochgefahren ist bzw. welches Betriebsystem gerade gebootet ist, kann dies LabStatus für Sie leisten. Sie können in LabStatus Ihre Hosts und entsprechende Indikatoren definieren. Diese werden kontinuierlich ausgeführt. Darüber hinaus gibt es die Möglichkeit Aktionen zu konfigurieren, die Sie bei Bedarf auslösen können (z.B. Wake-On-Lan).

Hosts hinzufügen:

Damit LabStatus überhaupt etwas tun kann, müssen natürlich erst einmal Hosts definiert werden.

Um einen Host hinzuzufügen:

  1. Öffnen Sie den Einstellungen-Dialog
  2. Klicken Sie auf den Hinzufügen-Button (+)
  3. Geben Sie den Hostnamen, die IP-Adresse, Mac-Adresse, Benutzername und Password ein

Indikatoren hinzufügen:

Sie können Indikatoren definieren, indem Sie ein Shell-Kommando und einen Namen angeben. Der Rückgabewert des Shell-kommandos wird benutzt um den Status des Indikators anzuzeigen. Ein Wert von 0 steht hierbei für eine erfolgreiche Ausführung und wird durch einen grünen Indikator wiedergegeben. Ein Wert gr¨ßer als 0 wird entsprechend mit der Farbe Rot gekennzeichnet. Zu beachten ist: Der Zeichenkette für das Kommando wird vor der Ausführung intern die Zeichenkette '1>/dev/null 2>&1' angehängt um keine Ausgaben zu produzieren.

Um einen Indikator hinzuzufügen:

  1. Öffnen Sie den Einstellungen-Dialog
  2. Klicken Sie auf den Hinzufügen-Button (+)
  3. Vergeben Sie einen Namen und das entsprechende Shell-Kommando
    4. Beispiel: Wir erzeugen einen Indikator mit dem Namen 'PING' und geben folgendes Kommando an: 
    /sbin/ping -t 2 -c 1 %ipaddress%
    Sie können %ipaddress%, %macaddress%, %hostname%, %username% und %password% in Ihren Shell-Kommandos verwenden.

Aktionen hinzufügen:

Sie können auch Aktionen definieren. Diese werden bei Bedarf durch einen Klick auf den zugeordneten Button ausgelößt. Zu beachten ist: Der Zeichenkette für das Kommando wird vor der Ausführung intern die Zeichenkette '1>/dev/null 2>&1' angehängt um keine Ausgaben zu produzieren.

Um eine Aktion hinzuzufügen:

  1. Öffnen Sie den Einstellungen-Dialog
  2. Klicken Sie auf den Hinzufügen-Button (+)
  3. Vergeben Sie einen Namen und das entsprechende Shell-Kommando
    4. Beispiel: Wir erzeugen eine Aktion mit dem Namen 'WOL' und geben folgenden Kommando an: 
    /usr/local/bin/wol %macaddress%
    Sie können %ipaddress%, %macaddress%, %hostname%, %username% und %password% in Ihren Shell-Kommandos verwenden.
